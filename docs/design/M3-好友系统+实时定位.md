## 三、数据模型（3 张表）

### friends 表
```sql
CREATE TABLE friends (
    id          BIGSERIAL PRIMARY KEY,
    user_id     VARCHAR(32) NOT NULL,
    friend_id   VARCHAR(32) NOT NULL,
    status      VARCHAR(20) DEFAULT 'pending',  -- pending/accepted/blocked
    created_at  TIMESTAMP DEFAULT NOW(),
    updated_at  TIMESTAMP DEFAULT NOW(),
    UNIQUE(user_id, friend_id)
);
```

### friend_requests 表
```sql
CREATE TABLE friend_requests (
    id          BIGSERIAL PRIMARY KEY,
    from_user_id VARCHAR(32) NOT NULL,
    to_user_id   VARCHAR(32) NOT NULL,
    message     VARCHAR(100),
    status      VARCHAR(20) DEFAULT 'pending',  -- pending/accepted/rejected
    created_at  TIMESTAMP DEFAULT NOW()
);
```

### location_records 表
```sql
CREATE TABLE location_records (
    id           BIGSERIAL PRIMARY KEY,
    user_id      VARCHAR(32) NOT NULL,
    latitude     DOUBLE PRECISION NOT NULL,
    longitude    DOUBLE PRECISION NOT NULL,
    accuracy     FLOAT,
    battery      INTEGER,
    is_charging  BOOLEAN DEFAULT FALSE,
    recorded_at  TIMESTAMP DEFAULT NOW()
);
CREATE INDEX idx_location_user_time ON location_records(user_id, recorded_at DESC);
```

### geo_fences 表
```sql
CREATE TABLE geo_fences (
    id              BIGSERIAL PRIMARY KEY,
    user_id         VARCHAR(32) NOT NULL,
    name            VARCHAR(50),
    latitude        DOUBLE PRECISION NOT NULL,
    longitude       DOUBLE PRECISION NOT NULL,
    radius          INTEGER NOT NULL,           -- 米
    is_active       BOOLEAN DEFAULT TRUE,
    target_user_id  VARCHAR(32) NOT NULL,
    created_at      TIMESTAMP DEFAULT NOW()
);
```

### Redis 缓存
| Key | 说明 | TTL |
|------|------|-----|
| `location:{userId}` | 最新位置 JSON | 2小时 |
| `sharing:{userId}` | 位置共享开关 (true/false) | 永久（手动清除） |

## 五、核心实现细节

### 5.1 距离计算
使用 **Haversine 公式** 计算两点间距离（适用于短距离、精度高、不依赖外部服务）。

### 5.2 定时任务
| 任务 | 频率 | 说明 |
|---|---|---|
| 轨迹清理 | 每天凌晨 3:00 | 删除 7 天前的 `location_records` |
| 围栏检测 | 每分钟 | 遍历所有 active 围栏 → 获取目标最新位置 → 计算距离 → 超出半径 → Redis Pub/Sub 推送预警 |

### 5.3 高德地图集成策略
- **开发阶段**：地图区域使用灰色占位 Container（200×300px），中心显示"地图加载中..."
- **高德 Key 到位后**：替换为 `amap_flutter_map` Widget，所有接口签名不变

### 5.4 环境监听/摄像头查看
- 仅提供入口 UI 占位
- 后端返回 `{ code:0, message:"功能即将上线" }`
