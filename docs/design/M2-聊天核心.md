## 四、核心实现策略

### 腾讯IM集成（Mock模式）

开发阶段不集成真实腾讯IM SDK，采用Mock策略：

- **后端**：消息存储到PostgreSQL `messages` 表，会话存在 `sessions` 表
- **发送消息**：写入DB → 返回成功 → 通过Redis Pub/Sub通知在线用户
- **接收消息**：前端轮询（3秒间隔）或 SSE 长连接
- **已读回执**：更新 `messages.read_at` 字段
- **UserSig**：复用M1的Mock HMAC生成

腾讯IM Key到位后：替换为真实SDK，接口签名不变。

### 文件上传（Mock）

- 图片/视频/文件存储在本地磁盘 `uploads/` 目录
- 返回本地URL
- COS/OSS到位后替换为云端存储

## 五、数据模型

### sessions 表
```sql
CREATE TABLE sessions (
    id BIGSERIAL PRIMARY KEY,
    session_id VARCHAR(32) UNIQUE NOT NULL,
    type VARCHAR(20) NOT NULL,        -- single / group
    name VARCHAR(100),
    avatar VARCHAR(500),
    last_message TEXT,
    last_message_type VARCHAR(20),
    last_time TIMESTAMP,
    is_pinned BOOLEAN DEFAULT FALSE,
    is_muted BOOLEAN DEFAULT FALSE,
    mute_expire TIMESTAMP,
    burn_enabled BOOLEAN DEFAULT FALSE,
    burn_duration INTEGER,             -- 秒
    created_at TIMESTAMP DEFAULT NOW()
);
```

### session_members 表
```sql
CREATE TABLE session_members (
    id BIGSERIAL PRIMARY KEY,
    session_id VARCHAR(32),
    user_id VARCHAR(32),
    role VARCHAR(20) DEFAULT 'member', -- owner / admin / member
    joined_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(session_id, user_id)
);
```

### messages 表
```sql
CREATE TABLE messages (
    id BIGSERIAL PRIMARY KEY,
    message_id VARCHAR(32) UNIQUE NOT NULL,
    session_id VARCHAR(32),
    sender_id VARCHAR(32),
    type VARCHAR(20) NOT NULL,         -- text/voice/image/video/file/location/emoji/system
    content TEXT,
    media_url VARCHAR(500),
    media_duration INTEGER,            -- 语音/视频时长(秒)
    file_name VARCHAR(200),
    file_size BIGINT,
    reply_to VARCHAR(32),              -- 引用消息ID
    burn_at TIMESTAMP,                 -- 阅后即焚到期时间
    read_at TIMESTAMP,                 -- 已读时间
    status VARCHAR(20) DEFAULT 'sent', -- sent/delivered/read/revoked
    sent_at TIMESTAMP DEFAULT NOW()
);
```

## 六、功能逻辑与异常边界（L3 摘要）

| 功能 | 正常 | 异常 |
|---|---|---|
| 发送文字 | 输入→发送→上屏 | 网络断开→感叹号→重发 |
| 发送图片 | 选择→压缩(≤2MB)→上传→发送 | 上传失败→重试 |
| 发送语音 | 按住→录音→松开→发送 | 录音<1秒→取消 |
| 发送视频 | 录制→压缩→发送 | >15秒→提示限制 |
| 发送文件 | 选择→上传→发送 | >100MB→提示过大 |
| 撤回 | 长按→撤回→显示"已撤回" | >24小时→不显示撤回 |
| 阅后即焚 | 开启→选时长→发送→到期销毁 | 对方已读→到期销毁 |
| 音视频通话 | 发起→接听→计时→挂断 | 30秒无应答→自动挂断 |
| 群公告 | 发布→顶部显示 | 非管理员→不显示按钮 |
| 群禁言 | 设置→禁言 | 禁言期间发消息→提示 |

### 错误码（2xxx）

| 错误码 | 说明 |
|---|---|

| 2007 | 录音时间太短 |
