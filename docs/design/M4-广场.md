## 三、数据模型（2 张表）

### posts 表（动态）
```sql
CREATE TABLE posts (
    id           BIGSERIAL PRIMARY KEY,
    post_id      VARCHAR(32) UNIQUE NOT NULL,
    user_id      VARCHAR(32) NOT NULL,
    content      TEXT,                           -- 文字内容（140字上限）
    media_type   VARCHAR(20) DEFAULT 'text',     -- text/image/video
    media_urls   TEXT,                            -- JSON数组，图片/视频URL列表
    media_duration INTEGER,                       -- 视频时长(秒)
    location     VARCHAR(100),                    -- 位置名称
    visibility   VARCHAR(20) DEFAULT 'public',   -- public/friends/private
    status       VARCHAR(20) DEFAULT 'published', -- published/draft/deleted
    like_count   INTEGER DEFAULT 0,
    comment_count INTEGER DEFAULT 0,
    share_count  INTEGER DEFAULT 0,
    created_at   TIMESTAMP DEFAULT NOW(),
    updated_at   TIMESTAMP DEFAULT NOW()
);
CREATE INDEX idx_posts_user ON posts(user_id, created_at DESC);
CREATE INDEX idx_posts_visibility ON posts(visibility, created_at DESC);
```

### comments 表（评论）
```sql
CREATE TABLE comments (
    id           BIGSERIAL PRIMARY KEY,
    comment_id   VARCHAR(32) UNIQUE NOT NULL,
    post_id      VARCHAR(32) NOT NULL,
    user_id      VARCHAR(32) NOT NULL,
    parent_id    VARCHAR(32),                    -- 父评论ID（回复用）
    reply_to_user VARCHAR(32),                   -- 被回复用户ID
    content      TEXT NOT NULL,
    like_count   INTEGER DEFAULT 0,
    created_at   TIMESTAMP DEFAULT NOW()
);
CREATE INDEX idx_comments_post ON comments(post_id, created_at ASC);
```

### likes 表（点赞）
```sql
CREATE TABLE likes (
    id          BIGSERIAL PRIMARY KEY,
    user_id     VARCHAR(32) NOT NULL,
    target_id   VARCHAR(32) NOT NULL,            -- post_id 或 comment_id
    target_type VARCHAR(20) NOT NULL,            -- post / comment
    created_at  TIMESTAMP DEFAULT NOW(),
    UNIQUE(user_id, target_id, target_type)
);
```

### follows 表（关注）
```sql
CREATE TABLE follows (
    id           BIGSERIAL PRIMARY KEY,
    follower_id  VARCHAR(32) NOT NULL,
    following_id VARCHAR(32) NOT NULL,
    created_at   TIMESTAMP DEFAULT NOW(),
    UNIQUE(follower_id, following_id)
);
```

## 四、后端接口（13 个）

### 动态接口
| API | 方法 | 路径 | 说明 |
|---|---|---|---|
| 获取动态列表 | GET | /api/v1/posts | ?feed=&page=&size= |
| 发布动态 | POST | /api/v1/posts | { content, mediaType, mediaUrls, location, visibility } |
| 获取动态详情 | GET | /api/v1/posts/{id} | — |
| 删除动态 | DELETE | /api/v1/posts/{id} | — |
| 点赞/取消 | POST | /api/v1/posts/{id}/like | — |
| 获取评论列表 | GET | /api/v1/posts/{id}/comments | ?page= |
| 发表评论 | POST | /api/v1/posts/{id}/comments | { content, parentId?, replyToUserId? } |
| 删除评论 | DELETE | /api/v1/comments/{id} | — |
