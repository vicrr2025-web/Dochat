## 二、数据模型

### 2.1 依赖现有表

| 表 | 使用字段 | 说明 |
|------|------|------|
| `users` | nickname, avatar, email, is_verified, credit_score, credit_level | 个人信息卡片展示 |
| `users` | password_hash | 修改密码校验 |

### 2.2 新表（2 张）

```sql
-- 设备管理表
CREATE TABLE user_devices (
    id BIGSERIAL PRIMARY KEY,
    device_id VARCHAR(64) NOT NULL UNIQUE,
    user_id VARCHAR(32) NOT NULL,
    device_name VARCHAR(128),
    device_model VARCHAR(64),
    os_version VARCHAR(32),
    is_current BOOLEAN DEFAULT FALSE,
    last_active_at TIMESTAMP NOT NULL DEFAULT NOW(),
    created_at TIMESTAMP NOT NULL DEFAULT NOW()
);
CREATE INDEX idx_user_devices_user ON user_devices(user_id);

-- 隐私设置表
CREATE TABLE user_privacy (
    id BIGSERIAL PRIMARY KEY,
    user_id VARCHAR(32) NOT NULL UNIQUE,
    online_visibility VARCHAR(16) DEFAULT 'all',     -- all / friends / none
    avatar_visibility VARCHAR(16) DEFAULT 'all',
    bio_visibility VARCHAR(16) DEFAULT 'all',
    message_permission VARCHAR(16) DEFAULT 'all',
    updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);
```

### 2.3 Redis 键

| Key | 类型 | TTL | 说明 |
|------|------|------|------|
| `device:{userId}:{deviceId}` | String | 30天 | 设备会话信息 |
