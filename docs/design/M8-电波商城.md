## 二、数据模型（6 表）

```sql
CREATE TABLE mall_products (
    id BIGSERIAL PRIMARY KEY,
    product_id VARCHAR(32) NOT NULL UNIQUE,
    seller_id VARCHAR(32) NOT NULL,
    shop_id VARCHAR(32),
    category VARCHAR(16) NOT NULL,     -- farm/factory/idle
    title VARCHAR(256) NOT NULL,
    description TEXT,
    price DECIMAL(10,2) NOT NULL,
    images TEXT,                        -- JSON array
    stock INT DEFAULT 1,
    status VARCHAR(16) DEFAULT 'active',
    recycle_estimate DECIMAL(10,2),     -- 回收估价
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE mall_cart_items (
    id BIGSERIAL PRIMARY KEY,
    user_id VARCHAR(32) NOT NULL,
    product_id VARCHAR(32) NOT NULL,
    quantity INT DEFAULT 1,
    selected BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(user_id, product_id)
);

CREATE TABLE mall_orders (
    id BIGSERIAL PRIMARY KEY,
    order_id VARCHAR(32) NOT NULL UNIQUE,
    buyer_id VARCHAR(32) NOT NULL,
    seller_id VARCHAR(32) NOT NULL,
    product_id VARCHAR(32) NOT NULL,
    amount DECIMAL(10,2) NOT NULL,
    quantity INT DEFAULT 1,
    status VARCHAR(16) DEFAULT 'pending',
    tracking_no VARCHAR(64),            -- 物流单号
    session_id VARCHAR(32),             -- 内嵌聊天
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE mall_reviews (
    id BIGSERIAL PRIMARY KEY,
    review_id VARCHAR(32) NOT NULL UNIQUE,
    order_id VARCHAR(32) NOT NULL,
    reviewer_id VARCHAR(32) NOT NULL,
    rating INT CHECK(rating BETWEEN 1 AND 5),
    content TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE mall_shops (
    id BIGSERIAL PRIMARY KEY,
    shop_id VARCHAR(32) NOT NULL UNIQUE,
    owner_id VARCHAR(32) NOT NULL,
    shop_name VARCHAR(128) NOT NULL,
    shop_logo VARCHAR(256),
    status VARCHAR(16) DEFAULT 'pending', -- pending/approved/rejected
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE mall_coupons (
    id BIGSERIAL PRIMARY KEY,
    coupon_id VARCHAR(32) NOT NULL UNIQUE,
    title VARCHAR(128),
    discount DECIMAL(10,2),
    min_amount DECIMAL(10,2),
    total_count INT,
    used_count INT DEFAULT 0,
    expires_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);
```

### 辅助表（复用）

| 表 | 用途 |
|------|------|
| `mall_favorites` | user_id + product_id，唯一约束 |
| `mall_browse_history` | user_id + product_id + viewed_at |
| `mall_disputes` | 复用 M7 `guarantee_disputes` 结构，参数不同：jury_count=17, threshold=9（M7 为 jury_count=19, threshold=10） |
