## 三、执行步骤

### 步骤 1：安装 Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

**验证：** `brew --version`

### 步骤 2：安装 JDK 17

```bash
brew install openjdk@17
# 配置环境变量到 ~/.zshrc
echo 'export PATH="/opt/homebrew/opt/openjdk@17/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

**验证：** `java --version` 输出 `17.x`

### 步骤 3：安装 Flutter SDK

```bash
# 下载 Flutter SDK
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable
# 配置环境变量
echo 'export PATH="$HOME/development/flutter/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

**验证：** `flutter doctor` 检查所有项

### 步骤 4：安装 CocoaPods

```bash
brew install cocoapods
```

**验证：** `pod --version`

### 步骤 5：安装 Android Studio

```bash
brew install --cask android-studio
# 首次启动后需完成 SDK 安装向导（手动）
```

**验证：** `flutter doctor --android-licenses` 通过

### 步骤 6：安装 PostgreSQL 16

```bash
brew install postgresql@16
brew services start postgresql@16
```

**验证：** `psql --version` 输出 `16.x`，`psql -U claude postgres` 可连接

### 步骤 7：安装 Redis 7

```bash
brew install redis
brew services start redis
```

**验证：** `redis-cli ping` 返回 `PONG`

### 步骤 8：安装 Maven

```bash
brew install maven
```

**验证：** `mvn --version`

### 步骤 9：安装 IntelliJ IDEA CE

```bash
brew install --cask intellij-idea-ce
```

**验证：** 可从 Applications 启动
